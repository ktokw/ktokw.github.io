# AI 시대 생존 테스트

> 당신은 AI 시대에서 살아남을 수 있을까요?

MBTI는 성격을 봤고, 이 테스트는 **생존 전략**을 본다.

20문항 · 약 3분 · 16가지 유형

## 바로가기

**[테스트 시작하기 →](https://ktokw.github.io/)**

## 유형 구조

4개 축 × 2극 = 16가지 생존 유형

| 축 | 질문 | A극 | B극 |
|---|---|---|---|
| AI 입성 시기 | 언제 뛰어드는가? | P (개척자) | V (검증자) |
| AI 활용 방식 | 얼마나 깊이 파는가? | D (탐구형) | U (실용형) |
| AI 시대 나침반 | 어디를 향하는가? | O (기회형) | G (수호형) |
| AI 활용 표현 | 어떻게 드러내는가? | S (공유형) | I (내재형) |

## 기술 스택

- 순수 HTML + CSS + Vanilla JS (프레임워크 없음)
- 단일 파일 (`index.html`) 완결
- GitHub Pages 배포
